# Animated Web App

A simple web application with animations created using HTML, CSS, and JavaScript. This app includes a bouncing ball, a rotating square, and a fading text effect. Additionally, the colors of these elements change randomly every second.

## Description

This Animated Web App demonstrates how to create dynamic and interactive animations using basic web technologies. It serves as a fun and educational example for those learning about web development and CSS animations.

## Features

- Bouncing ball animation
- Rotating square animation
- Fading text effect
- Random color changes for animated elements
